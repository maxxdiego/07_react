COMO RODAR O PROJETO BAIXADO
Instalar todas as dependências indicadas pelo package.json
### npm install

Rodar o projeto React
### npm run dev

Acessar no navegador
### http://localhost:3000


SEQUÊNCIA PARA CRIAR UM PROJETO REACT
### npx create-next-app

Acessar o diretório do PROJETO
### cd [nome-da-pasta]

Rodar o projeto React
### npm run dev

Acessar no navegador
### http://localhost:3000
