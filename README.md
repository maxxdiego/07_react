## Como executar

Pré-requisitos: 
- Visual Studio Code
- GIT

```bash
# clonar repositório
git clone https://github.com/maxxdiego/07_react.git

```

Instalar todas as dependências indicadas pelo package.json:

```bash
npm install

```

Rodar o projeto React
```bash
npm run dev

```

No navegador acessar as aplicações através do localhost:3000.
